# Event Planning System

## Product Summary
Event Planning System is a Notion workspace for planning events with tasks, vendors, budget, timeline, guests, and event details in one organized place.

## Main Audience
- Event planners
- Wedding planners
- Hosts
- Organizers
- Small teams planning structured events
- People planning events who want one clean command center

## Core Problem
Event planning gets messy when details are spread across notes, chats, spreadsheets, memory, and separate task lists.

## Main Promise
Plan the whole event from one clean Notion dashboard.

## What It Includes
- Event dashboard
- Task tracker
- Vendor tracker
- Budget tracker
- Timeline
- Guest/details area
- Organized planning views

## Best Positioning
A complete Notion event planning workspace for organizing vendors, tasks, budget, timeline, and event details.

## Buyer Pain Points
- Event details are scattered
- Vendors are hard to track
- Budget gets messy
- Timeline is unclear
- Tasks are easy to forget
- Planning feels stressful and unorganized

## CTA Ideas
- Get the Event Planning System
- View the Notion template
- Plan your event in one workspace
- Organize your event from start to finish
